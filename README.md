# 🎨 Frontend Styling Practice

A structured practice repository for improving frontend styling skills using **CSS, Bootstrap, and Tailwind CSS**.  
This repository contains exercises, UI components, and mini-projects designed to progress from **fundamental concepts to advanced layouts**.

The aim of this repository is to strengthen practical skills in **modern web styling, responsive design, and component-based UI development**.

---

## 🚀 Technologies Used

- **CSS3**
- **Bootstrap 5**
- **Tailwind CSS**

---

## 📚 Learning Path

### 1️⃣ CSS Fundamentals
- Selectors
- Box Model
- Flexbox
- Grid Layout
- Positioning
- Responsive Design
- Animations & Transitions

### 2️⃣ Bootstrap
- Grid System
- Utility Classes
- Components (Navbar, Cards, Buttons)
- Forms
- Responsive Layouts
- Bootstrap Customization

### 3️⃣ Tailwind CSS
- Utility-First Styling
- Layout Utilities
- Responsive Utilities
- Component Building
- Dark Mode
- Advanced Layouts
